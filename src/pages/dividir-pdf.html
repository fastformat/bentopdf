Dividir PDF

Extraia um intervalo de páginas para um novo PDF.

Clique para selecionar um arquivo ou arraste e solte

A single PDF file

Seus arquivos nunca saem do seu dispositivo.

Como funciona:

  • Digite os Números de Página separados por vírgulas (ex.: 2, 8, 14).
  • Digite intervalos de páginas usando um hífen (ex.: 5-10).
  • Combine-os para seleções complexas (ex.: 1-3, 7, 12-15).

Como funciona:

  • Extraia todas as páginas pares (2, 4, 6...) ou todas as páginas ímpares (1, 3, 5...) para um novo PDF.

Como funciona:

  • Cada página do PDF será salva como um arquivo PDF separado.
  • O resultado será baixado como um arquivo ZIP contendo todas as páginas.

Como funciona:

  • Clique nas miniaturas das páginas abaixo para selecionar as páginas que você quer extrair.
  • As páginas selecionadas serão destacadas.

Como funciona:

  • Divida o PDF com base em seus marcadores (sumário).
  • Selecione o nível de marcador em que dividir.

Como funciona:

  • Divida o PDF em vários arquivos, cada um contendo N páginas.

Saída

Vários arquivos são baixados juntos como um ZIP.

Perguntas Frequentes

Posso extrair intervalos de páginas específicos?

Sim! Você pode extrair páginas específicas ou intervalos de páginas do seu PDF. Selecione exatamente quais páginas você precisa ou divida em arquivos individuais.

A qualidade do PDF será perdida ao dividir?

Não! PDFs divididos mantêm exatamente a mesma qualidade do documento original. Não ocorre nenhuma compressão ou perda de qualidade.

Posso dividir múltiplos PDFs de uma vez?

Sim, processe vários PDFs em lotes para uma gestão de documentos eficiente.
